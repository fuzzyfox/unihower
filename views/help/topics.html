Create topics

How to

Navigate to the topic page and click on Create a topic

Once you have done that, you need to complete the form shown. Enter the name of your topic and then optionally enter a description.

Did you know: You can enter markdown to format your description

To finish, press .

Troubleshooting

Make sure you have included a name for your topic. This field is mandatory.

Checking it works

You should be taken to a new page that shows the name of your topic alongside an Eisenhower decision matrix. You should also have the option to edit or delete the topic.

You can also check the main topic page to see if your topic is listed.

Modify topics

How to

First navigate to the topic page and click on View Topic

Once you have done that, click Edit Topic

You can now make the changes to the name or description of the topic.

Remember: You can enter markdown to format your description

To finish, press

Troubleshooting

Make sure you have navigated to the correct topic and clicked Edit Topic

Also make sure you don't leave the name field blank.

Checking it works

You should be taken to a new page that shows the modified name or description of your topic. All your existing tasks will still be displayed on the matrix. You should also have the option to edit or delete the topic.

You can also check the main topic page to see if your modified topic is listed.

Topics

Danger Delete topics

How to

Note: This action is irreversible

First navigate to the topic page and click on View Topic

Double check you are about to delete the correct topic, click Delete Topic and accept the confirmation message.

Troubleshooting

Make sure you navigate to the correct topic.

If you have deleted a topic by mistake, unfortunately it cannot be recovered.

Checking it works

You will automatically be taken back to your topic page and the topic should no longer appear.
